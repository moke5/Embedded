### 开发版配网

/etc/profile

相当于.bashrc

```
# cd /IOT
# cd /
```



- 设置开发板的IP地址及网关配置

在ifconfig eth0 up后设置

```
ifconfig eth0 192.168.8.32
route add default gw 192.168.8.253 dev eth0
```



- 配置开发板的DNS服务器

/etc/resolv.conf

```c
vi /etc/resolv.conf
```



```c
nameserver 202.96.128.166

nameserver 202.96.128.133
    
search lan
```



```c
# 1. 先关闭eth0网卡
ifconfig eth0 down

# 2. 强制设置eth0为100M全双工（核心命令，解决协商不匹配）
ethtool -s eth0 speed 100 duplex full autoneg off

# 3. 重新启用eth0网卡
ifconfig eth0 up
```

### SSH

1.SSH_config.tar.ba下载到开发板根目录



2.进入开发板根目录，解压缩，配置协议

```c
cd /
tar -xzvf SSH_config.tar.bz
cd SSH/openssh/
./auto
```



3.配置ssh密码

```c
passwd root
```



4.重启开发板

```c
reboot
```



5.Ubuntu下载ssh服务

```shell
sudo apt install openssh-server
```



6.检查是否启动：

```shell
sudo ps -e | grep ssh
```



```shell
sudo service ssh stat
```
