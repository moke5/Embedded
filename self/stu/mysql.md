# sql

## mysql



##### 登录mysql

```shell
mysql -u username -p
```



##### 查看所有数据库

```sql
show databases;
```



##### 使用指定数据库

```sql
use databasesname;
```



##### 查看数据库表

```sql
show tables;
```



##### 查看表字段

```sql
desc tablename;
```



##### 查看数据

```sql
select * from tablename;
```
